# Stack Tecnológico

## Core
- React 18.3.1
- Vite 5.4.2
- React Router DOM 6.26.0

## Estilos
- Tailwind CSS 3.4.10
- Lucide React (iconos)

## Mapas
- Leaflet 1.9.4
- React-Leaflet 4.2.1

## Gráficos y Reportes
- Recharts 2.12.7
- jsPDF 2.5.1 + jspdf-autotable
- XLSX 0.18.5

## Utilidades
- date-fns 3.6.0
- clsx 2.1.1

**Why:** Stack moderno y liviano para demo funcional sin backend complejo.

**How to apply:**
- Usar componentes funcionales con hooks
- Tailwind para estilos (no CSS custom innecesario)
- Leaflet para mapas georreferenciados
- localStorage como persistencia principal
